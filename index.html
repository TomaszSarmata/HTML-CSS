The Basic Language of The Web: HTML

Posted by Laura Jones on Monday, June 21 st 2027

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and JavaScript. These are languages of the web.

In this post, let's focun on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stand for HyperText Markup Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTLM constists of elements that describe different typers of content: paragraphs, links, geadings ,images, videos, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts

  1. The opening tag
  2. The closing tag
  3. The actual element
